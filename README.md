# Student-performance-dashboard

Excel-based analytics dashboard designed to track and evaluate student performance using CGPA, attendance, study hours, and activity metrics. 

Implements PivotTables, PivotCharts, and conditional formatting for real-time visualization of academic trends and risk identification. 

Automates data processing using Excel formulas such as AVERAGE, COUNTIF, and validation rules to ensure accurate and clean data. 

Features interactive filtering and slicers to analyze individual and class-level performance dynamically. 

Includes forecasting and trend analysis to predict future student performance based on historical data patterns. 

Provides structured reports to support academic planning, mentoring, and performance improvement strategies.
